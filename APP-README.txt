

	Organizador de Biomas

------Sobre a aplicação---------

O objetivo é desenvolver uma aplicação que organize os animais do zológico
de acordo com o bioma mais adequado para cada espécie. Este trabalho é parte
do processo seletivo StartDB 2024.

------Funcionamento-------------

A aplicação foi planejada para funcionar da seguinte forma:

	1 - Apresentar ao usuário a lista de biomas disponíveis e
	as vagas livres respectivas;

	2 - Pedir ao usuário que insira o nome do animal que deseja
	alocar e o número de espécimes a serem alocados.

	3 - A aplicação então deverá cruzar informações para determinar
	se o animal pode ser alocado e se há vagas suficientes para ele.

	4 - Ao final, o programa perguntará ao usuário se deseja alocar
	outro animal. Caso não deseje, o programa fecha.
